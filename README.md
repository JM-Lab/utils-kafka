JMLab Utility Libraries For Kafka
=================================
## version
	0.1.0

## Installation

Checkout the source code:

    git clone https://github.com/JM-Lab/utils-kafka.git
    cd utils-kafka
    mvn install

## Usage
Set up pom.xml :

    (...)
    <dependency>
			<groupId>jmlab</groupId>
			<artifactId>jmlab-utils-kafka</artifactId>
			<version>0.1.0</version>
	</dependency>
    (...)
